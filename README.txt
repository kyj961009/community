커뮤니티 사이트

프로젝트 기간
▶2023.12.07 ~ 현재


개발 인원
▶ 1명(개인 프로젝트)

프로젝트의 목적
▶정보 공유를 목적으로 글을 작성하고 한눈에 작성된 글들을 보는 서비스

개발 환경
▶ SpringBoot(3.2.0), JPA
▶ JAVA8
▶ Gradle

테스트 환경
▶ JUnit4

구현 부분
▶ 메인페이지
	▷ 작성되어있는 글들을 볼 수 있어야함
	▷ 작성한 사람이 누군지 나와야함
	▷ 최초 작성 날짜 + 수정했을 시 그 날짜가 보여야함
	▷ 게시글을 작성 할 수 있어야함
	▷ 게시물을 보면 조회수가 올라가야함
▶작성된 글 페이지
	▷ 타이틀, 내용 작성자 작성되어있어야함
	▷ 작성된 글페이지에서 작성한 글을 수정할 수 있어야함
	▷ 댓글을 작성할 수 있어야함
	▷ 작성된 댓글들을 볼 수 있어야함
	▷ 글만 쓰는게 아닌 사진,파일을 업로드 할수 있어야함
▶ 댓글
	▷ 댓글이 작성되고 그 게시물로 다시 돌아가져야함
	▷ 댓글을 수정하고 삭제 할 수 있어야함
